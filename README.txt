README — QuartoSensorial (VR)

Projeto Unity: QuartoSensorial
Plataforma alvo: Android (Meta Quest)
Autor: Carlos André Tavora Soares — RESTIC 29

Objetivo
Protótipo de um quarto sensorial simples, com a intenção de evoluir para brinquedos/elementos funcionais voltados ao desenvolvimento de crianças autistas.

Como abrir
1) Abra o Unity Hub e selecione a pasta do projeto.
2) Abra a cena principal em: Assets/Scenes/QuartoSensorial.unity
   (se você estiver usando outra cena, abra a que estiver salva em Assets/Scenes)

Como testar no PC (Editor)
1) Clique em Play.
2) Use o Camera Rig da Meta (Building Blocks) para visualizar e navegar no ambiente.
Obs.: A navegação foi validada primeiro no Editor (PC), conforme requisito da atividade.

Conteúdo do ambiente
- Chão (Plane) com material aplicado
- Skybox configurado (Skybox/Procedural)
- Paredes do quarto
- Objetos principais: cama, caixa organizadora e arandelas

Repositório (entrega)
- Pastas incluídas: Assets, ProjectSettings, Packages
- Pastas ignoradas no Git: Library, Temp, Logs (via .gitignore)
